A modern Python project template with `uv`, `pre-commit`, and `pytest`.

## Installation

To install the package, run:

```shell
pip install changeme
```

## Usage

To use the CLI, run:

```shell
changeme --name "Your Name"
```

This will output:

```
Hello, Your Name!
```
